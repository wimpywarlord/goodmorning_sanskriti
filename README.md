### Installing

Download and install flutter on your device.
## Deployment

Clone the repository and open cmd in the cloned folder.

Open the andriod emulator or connect a andriod device to the pc.

Type in the command to verify the connection of an andriod device:

'''
flutter doctor
'''

Type in the command to run the application:

'''
flutter run 
'''

Enjoy the service of Goodmorning Girlfriend.
